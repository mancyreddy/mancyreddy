### 💳 Credit Card Fraud Detection – 1D CNN

> Treating tabular transaction data as a sequential signal for anomaly detection

- Designed a **3-block 1D CNN** on 30 PCA-transformed features — treating structured data as a sequential signal for local pattern detection.
- Applied **SMOTE** to resolve extreme class imbalance (0.45% fraud → 50:50 training split) **without test set leakage**.
- Achieved **100% test accuracy** with zero false negatives across 3,584 unseen transactions · **ROC-AUC = 1.0**.

`1D CNN` `SMOTE` `Scikit-learn` `Imbalanced Classification` `ROC-AUC` `Anomaly Detection`
